# 2minworkouts - Spotify Player

Select a playlist in Spotify and then run 2minworkouts.exe
Songs will play for 2.5 minute each. 


## Usage

This will play 24 songs (1hr workout)

./2minworkouts 

This will play 100 songs (very long workout)

./2minworkouts 100
